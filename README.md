## 简介

Profile.js 是一个前端的面试简历的页面框架。含有有趣的页面的交互。

## 配置

打开`src/profileconfig.json`文件，通过配置相关简历内容，页面会自动生成内容呈现的页面。

## Start

- Clone the repository

```
git clone https://github.com/BQXBQX/Profile.js.git

cd Profile.js

npm install
#OR
pnpm install

npm run dev
#OR
pnpm run dev
```

- Build

```
cd Profile.js

npm run build
#OR
pnpm run build
```
